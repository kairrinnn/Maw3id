<objective>
Build the LLM intent extraction layer: Zod schemas, system prompt builder, dual-model client wrappers (Gemini 2.5 Flash + GPT-4.1-mini), and the routing function that tries cheap first, escalates on low confidence, and falls back to unknown intent.

Purpose: BOT-02 requires a dual-model routing strategy. BOT-03 requires a guaranteed fallback when both models fail. This plan implements the extraction layer that the FSM (03-02) consumes and that the webhook (03-03) wires in.

Output: Four production files in `src/lib/llm/`, one test file, and two new npm dependencies installed.
</objective>

    Export `tryGemini(message: string, systemPrompt: string): Promise<IntentResult | null>`:
    ```typescript
    import { GoogleGenAI } from '@google/genai'
    import { zodToJsonSchema } from 'zod-to-json-schema'
    import { IntentResultSchema, type IntentResult } from './types'

    export async function tryGemini(message: string, systemPrompt: string): Promise<IntentResult | null> {
      try {
        const ai = new GoogleGenAI({ apiKey: process.env.GEMINI_API_KEY! })
        const result = await ai.models.generateContent({
          model: 'gemini-2.5-flash',
          contents: [{ role: 'user', parts: [{ text: message }] }],
          config: {
            systemInstruction: systemPrompt,
            responseMimeType: 'application/json',
            responseJsonSchema: zodToJsonSchema(IntentResultSchema),
          },
        })
        return IntentResultSchema.parse(JSON.parse(result.text ?? '{}'))
      } catch (err) {
        console.error('[llm] Gemini extraction failed:', err)
        return null
      }
    }
    ```

    **Step 2: Create `src/lib/llm/intent.ts`.**

    Export `extractIntent(message: string, state: ConversationState, services: Service[]): Promise<IntentResult>`:

    ```typescript
    import { buildSystemPrompt } from './prompts'
    import { tryGemini, tryOpenAI } from './models'
    import type { ConversationState, IntentResult, Service } from './types'

    const FALLBACK_INTENT: IntentResult = {
      intent: 'unknown', confidence: 'low',
      service_name: null, date_raw: null, time_raw: null,
    }

    export async function extractIntent(
      message: string,
      state: ConversationState,
      services: Service[]
    ): Promise<IntentResult> {
      // SKIP_LLM=true for testing without real API calls
      if (process.env.SKIP_LLM === 'true') {
        return FALLBACK_INTENT
      }

      const prompt = buildSystemPrompt(services, state.step)

      // Cheap model first
      const cheap = await tryGemini(message, prompt)
      if (cheap && cheap.confidence === 'high' && hasRequiredFields(cheap, state.step)) {
        return cheap
      }

      // Escalate
      const standard = await tryOpenAI(message, prompt)
      if (standard) return standard

      // Both failed
      return FALLBACK_INTENT
    }
    ```

    Implement `hasRequiredFields(intent: IntentResult, step: string): boolean` as a non-exported helper:
    - `awaiting_service` step requires `service_name` non-null
    - `awaiting_datetime` step requires at least one of `date_raw` or `time_raw` non-null
    - All other steps: return true (no extra required fields)
